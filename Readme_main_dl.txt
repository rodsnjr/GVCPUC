Utilização do programa main_dl.py

Dependências: opencv, sklearn, skimage, scipy, matplotlib e keras

Execução do program: python main_dl.py --dir diretorio_dos_csv --training nome_do_csv_training --testing nome_do_csv_testing 
Por exemplo: python main_dl.py --dir /home/daniel/GVCPUC/res/ --training door_e_indoor_training.csv --testing door_e_indoor_testing.csv

Diretório dos csv: .../GVCPUC/res/ 
	Os .csv precisam conter o diretório do Dataset. Por exemplo: Dataset/door_e_indoor/ 

Baixar e colocar o dataset no diretório do dataset: .../GVCPUC/res/Dataset/Nome_dataset
